# Music Listener App

Designed and developed a Music Listener App with features to browse and play music stored on the device.

Implemented dynamic music selection functionality, allowing users to browse and play music files from their device storage.

Integrated runtime permission requests to access and read external storage, ensuring a smooth and secure user experience.


## Technologies Used

-> Java

-> Android SDK

-> Android Studio


## Note

Ensure that you have granted the necessary permissions for the app to access your device's storage in order to fetch and play music files.
